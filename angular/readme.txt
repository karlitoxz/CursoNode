Angular: clase 01 - https://iberoamericana.mrooms.net/mod/lesson/view.php?id=267065

    instalar angular de forma global:
        npm install -g @angular/cli
    
    Crear un directorio y crear un app
        ng new my-app

    Moverse al directorio
        cd my-app
    
    Iniciar el servicio
        ng serve --open
